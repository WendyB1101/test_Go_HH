GET getSettings

Returns the current configuration of your instance — webhook URLs, message delays, notification toggles, and more.

GET getStateInstance

Returns the current authorization state of the instance — whether it is authorized, not authorized, or blocked.

POST sendMessage

Sends a text message to a personal or group chat. The message is queued and sent immediately if the instance is authorized.

POST sendFileByUrl

Sends a file from an external URL. Supports images, video, audio, and documents up to 100 MB.
